operadores especiais de atribuição:
#-----|OPERADOR|-----|EXEMPLO|-----|EQUIVALENTE|---
#-----|  +=    |-----| x+=1  |-----| x = x+1   |---
#-----|	-=     |-----| x-=1  |-----| x = x-1   |---
#-----|  *=    |-----| x*=2  |-----| x = x*2   |---
#-----|  /=    |-----| x/=2  |-----| x = x/2   |---
#-----| **=    |-----| x**=2 |-----| x = x**2  |---
#-----| //=    |-----| x//=2 |-----| x = x//2  |---
